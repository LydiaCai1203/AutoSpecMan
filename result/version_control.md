# Git 工作流习惯

## 分支策略

- **主分支: `main`**

## Commit 信息规范

- **格式: Conventional Commits (`type(scope): subject`)**
- **主要类型: `fix`, `feat`, `chore`, `refactor`**

**示例**:
  - `fix(scope): description`
  - `feat(scope): description`
  - `chore(scope): description`

## 分支命名规范

- **命名模式: `type/description`**
